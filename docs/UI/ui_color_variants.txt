Purpose

Notes
==================================================

Best candidates to try next in code
- Variant 3 if we want the logo vibe but slightly less danger-red.
- Variant 5 if we want a calmer, more mature tool look.
- Variant 4 if we want stronger contrast and a more stylized interface.

Good UI mapping rule
- Neutral actions: graphite / charcoal
- Main actions: branded warm accent
- Dangerous actions: darker red than the main accent
- Tabs: muted by default, accented only when selected
- Inputs: mostly neutral, accent only on focus
